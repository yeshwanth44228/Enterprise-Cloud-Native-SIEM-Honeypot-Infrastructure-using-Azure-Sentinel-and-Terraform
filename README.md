# Enterprise Cloud-Native SIEM & Honeypot Infrastructure

## Overview

This project demonstrates deployment of a cloud-native Security Information and Event Management (SIEM) platform using Microsoft Sentinel and Azure.

A Windows Server honeypot is exposed to the internet and monitored using Microsoft Sentinel.

## Technologies

- Microsoft Azure
- Microsoft Sentinel
- Log Analytics
- Terraform
- KQL
- Windows Server

## Features

- Threat Detection
- Incident Response
- Threat Hunting
- MITRE ATT&CK Mapping
